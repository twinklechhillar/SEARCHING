# 🔍 Searching in Java

## 📌 Overview

Searching is the process of finding a specific element in a collection of data, such as an array. It is one of the most fundamental concepts in Data Structures and Algorithms (DSA).

This repository contains my Java implementations and practice problems for Searching algorithms.

---

## 📚 Topics Covered

### ✅ Linear Search

* Search an element
* Print index of the element
* Element not found
* First occurrence
* Last occurrence
* Count occurrences
* Practice problems

### ✅ Binary Search

* Binary Search algorithm
* Search in a sorted array
* First occurrence
* Last occurrence
* Binary Search practice problems
